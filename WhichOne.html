WHICH ONE

Ponto de Venda de Aplicação WEB

A Which One é uma aplicação Web que permite escolher, entre várias opções, na compra de um produto, serviço ou outro bem que possa ser adquirido.

Tem como objetivo receber uma requisição e devolver uma resposta à solicitação do cliente de acordo com as características solicitadas.

O Which One não exige que o cliente crie uma estrutura ou necessite de profissionais para utilizar a aplicação, permitindo que se foque meramente na sua melhor opção. Permite uma abordagem gradual de implementação, podendo começar com um sistema mais básico e expandir conforme a rentabilidade das compras que consegue fazer ao comparar produtos. A implementação pode ser feita mesmo para os utilizadores mais básicos dando facilidade de acesso para todo o tipo de utilizadores.

A aplicação que temos disponível Which One é uma modalidade de serviço de software, distribuída pela internet, cuja principal característica é o baixo custo. 













PRODUTOS

OPÇÕES


Com a Which One tem a possibilidade de escolher um produto, bem ou serviço (PBS), com as melhores características. Quando o cliente insere os dados acerca do que pretende comparar, é-lhe dada uma resposta de forma simples e organizada, para conseguir uma análise rápida e intuitiva do produto que quer adquirir, sem grandes dificuldades.

Os PBS depois de serem introduzidos na aplicação, podem ser consultados através de uma lista de 10 produtos por página, dando acesso aos seguintes dados: nome, preço, data em que foi atualizada pela última vez e imagem associada.

A aplicação tem disponível uma comparação através de um ponto de dados que permite ser verificada através de diversas opções(numérico superior, numérico inferior, texto e data).

O melhor exemplo em que se pode verificar uma destas situações de características onde um valor maior neste caso é melhor, é por exemplo, através da memória RAM de um computador, ou no caso inverso, na comparação de preços.
